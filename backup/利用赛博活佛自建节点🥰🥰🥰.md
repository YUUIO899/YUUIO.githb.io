📣 edgetunnel | Cloudflare Worker 自建 VLESS 节点 支持多客户端订阅与转换


📱 简介：edgetunnel 是一个运行在 Cloudflare Workers 上的 VLESS 节点部署脚本，支持将 VLESS 配置信息转换为多客户端可直接使用的 订阅格式。用户可一键生成配置并在 Clash、Singbox、Surge、小火箭 等工具中导入使用，让节点部署与分发更高效、更自动化

　脚本已适配主流平台：
　• Windows / macOS：v2rayN、clash.meta、Clash Verge Rev 等
　• iOS：Surge、小火箭、SingBox (SFI)
　• Android：ClashMetaForAndroid、SingBox (SFA)
　其核心特性是基于 Workers Serverless 架构，无需服务器即可搭建轻量代理环境。项目仍在更新，支持通过视频教程快速上手部署
